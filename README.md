# Measure Model Lead — website

This is the Jekyll project behind measuremodellead.co.uk.

If you're setting this up for the first time, or you've forgotten how any of
it works, start with SETUP-GUIDE.md in this same folder. It walks through
creating the GitHub repository, uploading these files, turning on GitHub
Pages, and pointing the domain at it from GoDaddy.

Quick map of what's here:

- `index.md`, `about.md`, `white-paper.md` — the static pages
- `offers/` — the two offer pages
- `blog/` — the insights index page
- `_posts/` — individual long-form posts, one Markdown file per post
- `_layouts/`, `_includes/` — the page templates, you shouldn't need to
  touch these unless you're changing the design
- `assets/css/style.css` — all the visual styling in one file
- `assets/images/`, `assets/pdf/` — where your real images and the white
  paper PDF go

This file and SETUP-GUIDE.md are kept out of the published site.
